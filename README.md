# Job Marketplace App

A full-stack job and freelance marketplace platform built using the MERN stack, where users can post jobs, apply for jobs, and manage proposals.

## Features
- User authentication (JWT-based login/signup)
- Post a job
- Browse available jobs
- Apply to jobs
- Manage job proposals
- Role-based access (client/freelancer)

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT

## Folder Structure
backend/
  middleware/
  models/
  routes/
  server.js

## Getting Started
1. Clone the repository
2. Run `npm install` inside the backend folder
3. Create a `.env` file with your MongoDB URI and JWT secret
4. Run `node server.js` to start the server
